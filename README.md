# 🌍 TechRadar Globe
> Global Tech Intelligence Platform — Real-time IT trends, AI signals & datacenter energy efficiency on a 3D globe

## Features
- Real-time signals: arXiv / GitHub / HN / Dev.to / HuggingFace
- Datacenter energy efficiency (Green Score via Open-Meteo)
- Country hover intel brief
- Solar/Wind/Cooling efficiency visualization
- Military HUD aesthetic

## No API keys required

## Stack
Globe.gl · Open-Meteo · arXiv API · GitHub · HN Firebase API

## Run locally
```bash
python3 -m http.server 3001
open http://localhost:3001
```
